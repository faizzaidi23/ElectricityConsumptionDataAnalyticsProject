Plugging into the Future: An Exploration of Electricity Consumption Patterns Using Tableau
📊 Overview

"Plugging into the Future" is a data visualization project that analyzes electricity consumption patterns using Tableau. Through interactive dashboards and well-structured data, the project highlights important trends, seasonal variations, and regional differences in electricity usage. The goal is to present meaningful insights that help understand how electricity demand changes over time and across locations.

🧠 Objectives

Examine historical trends in electricity consumption.

Identify peak usage periods and possible influencing factors.

Compare electricity consumption across different regions.

Present actionable insights through interactive Tableau dashboards.

📁 Project Structure

Project 1.twbx — Tableau Packaged Workbook containing all dashboards and visualizations.

Cleaned_Consumption.csv — Processed dataset used for analysis and visualization, including attributes such as date, region, consumption, and related variables.

README.md — Documentation for the project.

📌 Data Source

The dataset used in this project (Cleaned_Consumption.csv) contains structured information about electricity consumption across different regions and time periods. The data was cleaned and preprocessed to remove inconsistencies and missing values, ensuring that it is suitable for analysis and visualization in Tableau.

🛠️ Tools & Technologies

Tableau Desktop for visualization and dashboard creation

Microsoft Excel / Python for data cleaning and preprocessing

CSV format for structured data storage

📈 Key Visualizations

Time-series charts displaying electricity consumption trends over months and years.

Regional comparison dashboards showing differences in electricity usage.

Heatmaps and visual analyses highlighting seasonal consumption patterns.

KPI indicators and interactive dashboards for deeper data exploration.

🚀 How to Use

Download the Project 1.twbx file.

Open the file using Tableau Desktop.

Navigate through the dashboards and apply filters to explore insights from the dataset.

✅ Features

Cleaned and structured electricity consumption dataset.

Interactive and user-friendly Tableau dashboards.

Filters and drill-down capabilities for detailed exploration.

Useful insights that can support data-driven decisions and energy policy planning.
